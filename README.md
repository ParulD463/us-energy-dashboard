# Overview
### This dashboard explores how the United States generates electricity and how the energy mix has transformed over the past two decades. Built using Power BI Desktop with publicly available data from the U.S. Energy Information Administration (EIA).

## Key Insights
1. Coal generation has fallen over 65% since 2001
2. Natural gas surpassed coal around 2015 and is now the leading source
3. Nuclear output has remained remarkably stable throughout
4. Wind generation has grown 67x since 2001
5. Solar barely existed in this dataset before 2014 and is now the fastest growing source

## Data Source
### U.S. Energy Information Administration (EIA)
### https://www.eia.gov/electricity/data/browser/

## Tools Used
- Power BI Desktop (free)
- Power Query for data cleaning and transformation
